Workbench Component: Ppspell

Ppspell runs an intelligent spellcheck on a text file. It uses a combination of filtering hueristics backed by the aspell dictionaries for English and several other languages.

If a single, unzipped file is uploaded a straightforward spell-check is completed. Suspect words are presented with line numbers and context. Users with an available project Good Word List can improve the accuracy of the check by creating a Zip file containing the text file and the good words in a file named good_words.txt. Ppspell will use a good_words.txt file if it finds one.

The good_words.txt file can be Latin-1 or UTF-8. However, if you want to include words with diacriticals not available in Latin-1, submit a UTF-8 good_words.txt file.

Currently the dictionary used is aspell's en_US. Others may be available based on demand.
